== Description ==
-------------------------------------------------------
Patricia Lite is WordPress theme made for bloggers who wants to share your travel expertise, photography, fashion tips, food recipes, lifestyle, beauty and makeup, coaching, or any other niche blog. This theme has has beautiful featured slider, footer logo, 2 layout variation, custom widgets and many more including changing the color scheme to change the site color. Patricia Lite is mobile friendly, SEO optimized and loads faster. Check demo at https://volthemes.com/demo/?theme=patricia-lite

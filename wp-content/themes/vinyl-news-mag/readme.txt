== Description ==

Vinyl News Mag is a responsive multipurpose WordPress magazine website theme. It is a powerful platform for creating sophisticated magazine websites in a flash. Vinyl News Mag is a blog and magazine-specialized WordPress theme. You will find a lot of customizable options on the theme. Vinyl News Mag is a WordPress theme with a strong focus on advertisement and publicity. Browse the demo : https://demo.elegantblogthemes.com/vinylnewsmag/
 



== Installation ==

1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload Theme and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.
